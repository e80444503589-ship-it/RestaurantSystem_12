Система Ресторан

Описание проекта:
Это консольное приложение, моделирующее работу ресторана по варианту 12 лабораторной работы.

Функциональность:
- Клиент может просмотреть меню и сформировать заказ, выбирая блюда.
- Администратор подтверждает заказ и отправляет его на кухню (симулируется изменением статуса).
- Администратор выставляет счёт на основе заказа.
- Клиент может оплатить счёт.

Технологии: 
Git для контроля версий
